# soundio-cr

This is Crystal bindings for [libsoundio](https://github.com/andrewrk/libsoundio).

## Installation

1. Install libsoundio as you like.

2. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     soundio-cr:
       github: astellon/soundio-cr
   ```

3. Run `shards install`

## Usage

```crystal
require "soundio"
```
